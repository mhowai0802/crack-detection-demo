# Sample Images

Drop 6-10 curated demo images here so the Streamlit app can show a "Try a
sample" picker even without uploads.

Suggested naming convention:

- `crack_01.jpg`, `crack_02.jpg`, ... for positive (cracked) examples
- `no_crack_01.jpg`, `no_crack_02.jpg`, ... for negative examples

Good sources:

- A handful of images taken from the held-out split of the Mendeley dataset
  (keep attribution in the project README).
- Real site photos from construction projects (with permission).
- Unsplash / Pexels search terms: `concrete crack`, `concrete wall`, `cracked
  pavement`.

Supported formats: `.jpg`, `.jpeg`, `.png`.
